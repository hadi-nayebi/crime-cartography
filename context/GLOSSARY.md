home in the three shared-context clusters; this file does not create alternate
definitions.

## How a term changes

1. A draft term is defined in its canonical cluster and linked to a substantive
   Discussion.
2. Public feedback and qualified review are summarized into a proposed revision.
3. Hadi consolidates the term or keeps it draft; the repository records the
   change before the website adopts it.
